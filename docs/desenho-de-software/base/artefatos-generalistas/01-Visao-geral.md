# Visão Geral da Elicitação e Design

## Introdução
<p align="justify">Esta seção consolida os artefatos e as diretrizes estratégicas estabelecidas durante a <b>Design Sprint</b>. O conjunto de documentos aqui apresentados fundamenta o escopo do projeto <b>EcoGame</b>, integrando requisitos técnicos e visões de negócio para garantir uma base sólida para as etapas subsequentes de desenvolvimento e implementação.</p>

### Fluxo de Desenvolvimento
A definição do escopo fundamentou-se em instrumentos que asseguram a rastreabilidade e o alinhamento do projeto:

- [Glossário](../artefatos-generalistas/02-Glossario.md): Padronização de termos técnicos para uniformizar a comunicação entre a equipe de desenvolvimento e os stakeholders.

- [Mapa Mental](../artefatos-generalistas/03-Mapa-mental.md): Estruturação lógica das ideias e atividades, servindo como base para o levantamento preliminar de requisitos e a arquitetura inicial do jogo.

- [Rich Pictures](../artefatos-generalistas/04-Rich-pictures.md): Diagramação visual que permite a análise de diferentes perspectivas do sistema, auxiliando na identificação de conflitos e na compreensão global do fluxo do projeto.

- [Protótipo](../artefatos-generalistas/05-Prototipo.md): Representação visual de mecânicas básicas e interface, utilizada para validar os requisitos e antecipar a viabilidade das futuras implementações.
