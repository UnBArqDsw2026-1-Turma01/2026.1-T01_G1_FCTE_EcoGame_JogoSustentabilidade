# Validação do Design Sprint: Entrevistas

## Introdução
<p align="justify">
Esta página reúne os registros das entrevistas realizadas para a validação do <b>Design Sprint 05</b>. O objetivo dessas sessões foi coletar <i>feedbacks</i> reais de usuários sobre o protótipo desenvolvido, permitindo ajustes rápidos e garantindo que as mecânicas de jogo atendam às expectativas do público-alvo.
</p>
